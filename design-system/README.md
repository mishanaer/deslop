# Design System

Единая установка дизайн-системы Deslop для веб-продуктов и Telegram Mini Apps.

```bash
npx @deslop/design-system setup
```

Команда предложит выбрать тип продукта, затем подключит Primitives, нужные
компоненты, правила для агента и локальную проверку перед коммитом.

Без интерактивного выбора:

```bash
npx @deslop/design-system setup --web
npx @deslop/design-system setup --mini-app
```
