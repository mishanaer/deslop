# Design System

Единая установка дизайн-системы Deslop для веб-продуктов и Telegram Mini Apps.

## Что входит

| Пакет | Для чего нужен | Выбор при установке |
| --- | --- | --- |
| [Mini App](../mini-app/README.md) | React-компоненты для интерфейсов внутри Telegram: навигация, формы, списки, модальные окна и другие мобильные сценарии. | `--mini-app` |
| [Web UI](../web-ui/README.md) | React-компоненты для сайтов и веб-приложений: формы, меню, таблицы, диалоги, графики и готовые блоки. | `--web` |
| [Primitives](../primitives/README.md) | Общие цвета, типографика, отступы, радиусы, шрифты и иконки. Подключаются автоматически вместе с выбранной библиотекой компонентов. | Автоматически |

## Установка

```bash
npx @deslop/design-system setup
```

Команда предложит выбрать тип продукта, затем подключит Primitives, нужные
компоненты, правила для агента и локальную проверку перед коммитом.

Без интерактивного выбора:

```bash
npx @deslop/design-system setup --web
npx @deslop/design-system setup --mini-app
```
