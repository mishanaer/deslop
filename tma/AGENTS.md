# TMA

- Используй Tailwind CSS и токены из `src/styles/tailwind.css`. Не добавляй новые CSS Modules; существующие `*.module.css` — переходный слой.
- Для интерфейса переиспользуй существующие компоненты TMA и токены, шрифты и иконки `@deslop/primitives`. Перед созданием нового компонента проверь `src/components/`; не создавай локальный дубликат.
- Для Telegram используй только `@lib/twa`. Telegram-функции должны безопасно работать в обычном браузере.
- Перед завершением изменений в коде запусти из корня репозитория `npm run verify`.
