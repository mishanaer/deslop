## Web UI

- Перед изменением интерфейса прочитай `agent/COMPONENTS.md` и выбери существующий компонент по назначению.
- Работай с компонентами в `src/components`; не создавай параллельную gateway-копию и не подключай другую UI-библиотеку.
- Для кнопок, полей, переключателей, таблиц, меню, модалок, навигации и обратной связи используй соответствующий компонент Web UI, а не сырой HTML-контрол.
- Используй токены, шрифты и иконки из соседнего `../primitives` через алиас `@deslop/primitives`. Не добавляй хардкод цветов, размеров, отступов, радиусов, теней, локальные SVG, inline styles или Tailwind arbitrary values.
- Если нужного компонента нет, добавь его в Web UI на основе Primitives, не создавая продуктовый дубликат.
- Перед завершением запускай `pnpm verify` из `web-ui`.
