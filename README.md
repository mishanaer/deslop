# deslop

В репозитории лежат скилл [sasha](./sasha/SKILL.md), общие [Primitives](./primitives/README.md), библиотека компонентов [TMA](./tma/README.md) и инструкции для агентов [AGENTS.md](./AGENTS.md).

## Primitives

[Primitives](./primitives/README.md) — единый набор токенов, шрифтов и иконок для продуктов. Готовые React-компоненты и Storybook находятся в [tma](./tma/README.md), пакете `@deslop/tma`. Переносимый скилл для агентов находится в [skills/deslop-design-system](./skills/deslop-design-system/SKILL.md).

## sasha

sasha — скилл для ИИ-агентов, который помогает писать, проверять и улучшать русские тексты для интерфейсов. Внутри — правила для компонентов и ошибок, примеры, типографика и скрипты для поиска и проверки UI-текстов.

## AGENTS.md

`AGENTS.md` задаёт общие правила работы агента: как общаться, когда действовать самостоятельно, как менять код и проверять результат.

## Локальная проверка

На чистом клоне один раз установите зависимости TMA:

```bash
corepack yarn --cwd tma install --immutable
```

Затем перед коммитом запускайте из корня репозитория:

```bash
npm run verify
```

Команда проверяет токены, типографику и иконки Primitives, а также линт и
сборку Storybook и библиотеки TMA. Она не требует GitHub Actions и не меняет
исходные файлы.
