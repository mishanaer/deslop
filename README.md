# deslop

Общий репозиторий дизайн-системы и инструментов для её применения.

## AGENTS.md

[AGENTS.md](./AGENTS.md) задаёт правила для агентов: как использовать
дизайн-систему, когда действовать самостоятельно и как проверять изменения.

## primitives

[primitives](./primitives/README.md) — набор общих элементов дизайна: цветов,
шрифтов, отступов, скруглений и иконок. Благодаря ему интерфейсы в разных
продуктах выглядят единообразно.

## TMA

[TMA](./tma/README.md) — библиотека готовых React-компонентов на основе
primitives. Внутри — компоненты, Storybook и интеграция с Telegram Mini Apps,
которая безопасно работает и в обычном вебе. Пакет: `@deslop/tma`.

## sasha

[sasha](./sasha/SKILL.md) — скилл для ИИ-агентов, который помогает писать,
проверять и улучшать русские тексты интерфейса.
